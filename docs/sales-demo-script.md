# Sales Demo Script — De Raedt Growth OS (5–7 min)

## 0) Opening (15 sec)
"Dit is geen website. Dit is een groei‑ en control platform dat administratie verlaagt en opportuniteiten sneller omzet in opdrachten."

## 1) Procurement Hub (60 sec)
- Toon certificaten & erkenningen
- Klik "Tender Pack aanvragen"
- Toon dat procurement info centraal staat (geen mail-zoektocht)

## 2) Intake flows (60 sec)
- Start projectplanner (project intake)
- Toon facility interventie flow (met foto/urgentie)
- Leg uit: "dit wordt automatisch een record met owner + status"

## 3) Werken bij (60 sec)
- Toon job detail + apply
- Leg uit: "public site is single source of truth; later VDAB publishing adapter"

## 4) Admin Cockpit (90 sec)
- Login als directie
- Toon: tender pipeline + deadlines + expiry radar + leads + recruitment funnel
- "Dit is jullie dagelijkse stuurdashboard"

## 5) Tender Hub (90 sec)
- Toon nieuwe tenders ingest (TED)
- Go/no-go checklist
- Deadline reminders + dossier pack export

## 6) Afsluiter (15 sec)
"Jullie winnen tijd, zien sneller kansen, en sturen op cijfers. Dat is waarom dit een investering is (setup + retainer)."
